# 音乐播放器使用说明

## 直接使用方法

由于Node.js环境未安装，我已经将音乐播放器修改为纯HTML版本，您可以直接使用：

1. 双击打开 `index.html` 文件（会在默认浏览器中运行）
2. 点击「添加文件」按钮选择本地音乐文件
3. 点击播放列表中的歌曲开始播放

## 功能介绍

- **播放控制**：播放/暂停、上一首、下一首
- **进度控制**：拖动进度条快速跳转
- **音量调节**：可调节音量大小
- **播放列表管理**：添加音乐文件、清空列表
- **右键菜单**：在播放列表项上右键可选择播放或移除

## 注意事项

- 由于浏览器安全限制，无法直接访问本地文件系统中的文件夹
- 每次刷新页面后需要重新选择音乐文件
- 支持的音乐格式：mp3、wav、flac、aac、ogg、wma

## 打包为可执行文件（exe）的方法

如果您希望将此播放器打包为独立的可执行文件，有以下几种方法：

### 方法一：使用Electron（需要安装Node.js）

1. 安装Node.js（从官网 https://nodejs.org/ 下载并安装）
2. 打开命令行，进入本目录
3. 执行以下命令：
   ```
   npm install
   npm run build
   ```

### 方法二：使用NW.js（更简单的方式）

1. 从NW.js官网（https://nwjs.io/）下载适合您系统的SDK版本
2. 解压下载的文件
3. 将本目录中的所有文件复制到NW.js解压后的目录
4. 创建一个名为 `package.json` 的文件（如果没有），内容如下：
   ```json
   {
     "name": "music-player",
     "main": "index.html",
     "window": {
       "title": "音乐播放器",
       "width": 800,
       "height": 600
     }
   }
   ```
5. 双击运行 `nw.exe` 即可启动应用
6. 您可以使用Enigma Virtual Box等工具将NW.js和您的应用打包为单个可执行文件

### 方法三：使用WebView2（Windows平台）

1. 下载Microsoft Edge WebView2 Runtime（如果尚未安装）
2. 使用Visual Studio创建一个基于WebView2的桌面应用程序
3. 将本目录中的所有文件包含到项目中
4. 设置WebView2加载index.html
5. 编译生成可执行文件

## 常见问题

- **无法播放某些音乐文件**：请检查文件格式是否受支持，或文件是否损坏
- **播放列表不保存**：浏览器版本由于安全限制，无法保存完整的播放列表信息
- **音量调节不生效**：请检查系统音量设置

如有其他问题，请尝试在不同的浏览器中打开使用，如Chrome、Firefox或Edge。
