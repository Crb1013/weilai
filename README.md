# To-Do List 应用

这是一个使用纯 HTML、CSS 和 JavaScript 构建的简单而功能完整的待办事项应用。

## 功能特性

- ✅ 添加新的待办事项
- ✅ 标记待办事项为已完成/未完成
- ✅ 删除待办事项
- ✅ 按状态筛选待办事项（全部、进行中、已完成）
- ✅ 清除所有已完成的待办事项
- ✅ 本地存储功能（刷新页面后数据不会丢失）
- ✅ 响应式设计，适配各种设备屏幕
- ✅ 简洁美观的用户界面

## 如何使用

1. 克隆或下载此项目到本地
2. 在浏览器中打开 `index.html` 文件即可使用
3. 无需安装任何依赖

## 项目结构

```
├── index.html  # 应用的 HTML 结构
├── style.css   # 应用的样式
├── app.js      # 应用的核心功能逻辑
└── README.md   # 项目说明文档
```

## 技术栈

- **HTML5**: 用于构建应用的结构
- **CSS3**: 用于样式设计和响应式布局
- **JavaScript**: 用于实现应用的交互功能
- **localStorage**: 用于在本地存储待办事项数据

## 使用指南

1. **添加待办事项**: 在输入框中输入内容，然后点击"添加"按钮或按回车键
2. **标记完成状态**: 点击待办事项前面的复选框
3. **删除待办事项**: 点击待办事项右侧的"删除"按钮
4. **筛选待办事项**: 使用顶部的筛选按钮查看不同状态的待办事项
5. **清除已完成**: 点击底部的"清除已完成"按钮删除所有已完成的待办事项
